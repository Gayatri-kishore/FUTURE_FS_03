# Cheese Burst Restaurant - Task 3
### Future Interns Full Stack Web Development Internship (2026)

## 🎯 Project Objective
The goal of this project was to move beyond "imaginary" apps and build a functional, high-value website for a real local business: **Cheese Burst Restaurant**.

## 🚀 Key Features
- **Responsive Design:** Mobile-first layout for customers ordering on the go.
- **Dynamic Shopping Cart:** Users can add multiple items and see a live total.
- **WhatsApp Integration:** A custom-built checkout system that formats the cart into a direct WhatsApp message for the business owner.
- **Interactive Elements:** Functional mobile menu and floating support buttons.

## 🛠️ Tech Stack
- **HTML5** (Structure)
- **CSS3** (Custom Styling & Layout)
- **JavaScript** (Cart Logic & WhatsApp API Formatting)
